# Banking System - C++

This project demonstrates:

- ✔️ Classes & Objects  
- ✔️ Encapsulation (private data + getters)  
- ✔️ Inheritance (Savings, Checking, FixedDeposit)  
- ✔️ Polymorphism (calculateInterest() via base class pointer)  

## Features
- Deposit & Withdraw
- Account Info Display
- Overdraft Check
- Interest Calculation

## Technologies
- C++ (OOP Concepts)
